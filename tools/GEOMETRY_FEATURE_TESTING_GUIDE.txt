INSTALLATION
================================================================================

1. Extract the zip file:
   unzip vtol_analyzer_v4.1_geometry.zip

================================================================================
COMMON ISSUES & SOLUTIONS
================================================================================

ISSUE: "ModuleNotFoundError: No module named 'matplotlib'"
SOLUTION: pip install matplotlib

ISSUE: "ModuleNotFoundError: No module named 'tkinter'"
SOLUTION:
   - Ubuntu/Debian: sudo apt-get install python3-tk
   - macOS: Should be included with Python
   - Windows: Reinstall Python with tkinter option

ISSUE: Schematic appears blank or doesn't update
SOLUTION:
   - Check that parameters are valid (green checkmarks)
   - Click "Apply Changes" before "Update Schematic"
   - Verify matplotlib backend is working

ISSUE: Tooltips don't appear
SOLUTION:
   - Tooltips require hovering for 1 second
   - Make sure mouse is over the input field
   - Check window manager allows tooltips

================================================================================
TECHNICAL SPECIFICATIONS
================================================================================

Tail Fin Drag Model:
- Method: Wetted area with empirical corrections
- Form Factor: FF = 1 + 2*(t/c) + 60*(t/c)^4
- Interference: 1.05 (5% increase at junction)
- Skin Friction: cf = 0.008 (turbulent flow)
- Typical CD0 contribution: 0.0015-0.0020

Schematic Drawing:
- Format: Matplotlib figure (3 subplots)
- Resolution: Configurable DPI (default 100)
- Export: PNG, PDF, SVG supported
- Aspect ratio: Locked to true scale
- Dimensions: Automatically labeled

Parameter Ranges:
- Fuselage length: 0.8-2.0m (warn outside 0.8-2.0m)
- Fuselage diameter: 0.05-0.20m (warn outside 0.05-0.20m)
- Tail fin count: 3 or 4 only
- Tail fin chord: 0.03-0.10m (warn outside 0.03-0.10m)
- Tail fin span: 0.10-0.30m (warn outside 0.10-0.30m)
- Motor spacing: 0.3-0.8m (warn outside 0.3-0.8m)
