# Roadmap

MDS is evolving from a drone-show toolkit into a broader mission-directed swarm
stack for PX4/MAVLink fleets.

## Current Focus

- reliable SITL-to-real-hardware workflows
- operator-friendly fleet monitoring and command flows
- source-aware altitude and telemetry freshness
- environment/control-plane consistency
- Smart Wi-Fi Manager and MAVLink Anywhere sidecar integration
- clear docs, test gates, and deployment handoffs

## Near-Term

- final brand assets and social previews after logo approval
- richer field validation reports from real hardware and SITL
- continued Fleet Ops polish for sidecar profile status and reconcile flows
- optional sidecar dashboard/API auth design
- Caddy/reverse-proxy guidance for multi-dashboard deployments
- MCP-compatible AI-agent workflows for fleet inspection, planning, and command
  review
- automated SAR, surveillance, and cooperative mission templates
- PixEagle integration for vision-assisted swarm control
- optimized minimal SIH Docker simulation for lightweight validation

## Later

- encrypted fleet Wi-Fi and sidecar secrets
- stronger audit logs for fleet profile changes
- expanded AI/MCP agent-safe API workflows compatible with OpenClaw-style and
  other agent orchestration systems
- GPS-denied and indoor swarm feasibility using AI/image-processing pipelines
  such as monocular vision
- ArduPilot support exploration alongside the current PX4-first path
- additional mission templates for inspection, cooperative autonomy, and
  advanced air mobility research

MDS is not certified avionics. Roadmap items support research, testing, and
operator tooling; flight operations remain subject to local regulations and
independent safety review.
